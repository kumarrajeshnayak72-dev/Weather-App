Weather App 🌦️

A responsive Weather Application built using React that allows users to search for any city and view real-time weather information.

🚀 Features

- Search weather by city name
- Display current temperature
- Show humidity levels
- Show wind speed
- Responsive and user-friendly UI
- Real-time weather data using API

🛠️ Technologies Used

- React.js
- JavaScript (ES6+)
- CSS / Tailwind CSS
- Axios
- Weather API

🎯 Learning Outcomes

- Learned React state management using Hooks.
- Worked with APIs and asynchronous data fetching.
- Improved component-based architecture skills.
- Built a responsive user interface.

🔮 Future Improvements

- 5-day weather forecast
- Dark/Light mode
- Location-based weather detection
- Weather icons and animations
